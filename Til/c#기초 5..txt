c#기초 5.

1) 데이터 직렬화

데이터 구조나 객체 상태를 저장하고 나중에 재구성할 수 있는 포맷으로 변환하는 과정
객체를 파일의 형태로 저장하고 통신하기 쉬운 포맷으로 변환하는 과정

컴퓨터가 읽기 쉬운 파일 형태중 하나인 Json으로 통신함
>> Json 직렬화를 기본으로 함

C#과 유니티에서는 Newtonsoft.Json 패키지를 사용해서 변환함


2) Json 형식

[] 로 리스트, {} 로 객체를 표현
"키" : 값 - 형태를 지님

변환할 시에 JsonConvert.SerializeObject(직렬화)/DeserializeObject(역직렬화) 를 사용
